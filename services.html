Information Security Services

Web-Application Security Assessment
(WASA)

Fine-grained security Assessments addressed to Web Applications, with different levels of depth. Being all "budget-saving" solutions, it will be possible to resize the scan's type scenario (white-box, gray-box, black-box) according to the needs.

Web-Server Penetration Testing
(VAPT)

Architectural and infrastructural assessments. Also in this case, based on the possibilities, it will be possible to resize the offer on one or more realities and with different intrusion perspectives.

Compliance-based Assessments
(OWASP)

Checklist-compliant assessment services (eg OWASP, SANS Checklists).

Infrastructure Security Overview

We provide consultancy services regarding the infrastructural resources organization (firewalls, IDS, IPS, web servers, etc...) and also logic's architectural services (login architecture forms, session management, password policy, GDPR Compliant logs, etc ...).

Internal Security Audit
(Social Engineering)

For higher budgets, on-site assessments are also possible, in order to carry out audits by implementing social engineering attacks.

Information Security Training Course

On-site training courses are available in the field of cyber security (GDPR, password management, cryptographic algorithms, information security concepts, web application secure develpment, etc...).
